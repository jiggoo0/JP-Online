# 🎨 Tactical UI Excellence Protocol

Apply these rules to transcend basic UI and create a high-authority visual experience.

---

## 🛡️ 1. Executive Stealth Depth

- **Negative Space Mastery**: ใช้พื้นที่ว่างอย่างจงใจเพื่อสร้าง "ความขลัง" (High-end Discretion)
- **Glassmorphism v2**: ไม่ใช้แค่ Blur แต่ใช้การผสมผสานของ `backdrop-filter`, `mask-image`, และ `border-gradient` เพื่อสร้างมิติ
- **Obsidian Layering**: ใช้เฉดสีดำที่ต่างกัน (Obsidian, Charcoal, Deep Slate) เพื่อแยกความสำคัญของข้อมูลโดยไม่ใช้สีฉูดฉาด

## 📐 2. Fluid Precision Typography

- **Golden Ratio Scaling**: ใช้ตัวเลขสัดส่วนทองคำในการคำนวณ `line-height` และ `letter-spacing`
- **Fluid Sizing**: บังคับใช้ `clamp()` สำหรับ Headings เพื่อให้ขนาดสมบูรณ์แบบในทุกอุปกรณ์
- **Mono-Tactical**: ใช้ Monospace fonts เฉพาะจุดสำคัญ (IDs, Markers) เพื่อสื่อถึง "Technical Depth"

## ⚡ 3. Responsive Micro-Interactions

- **Data Scanning Effect**: เมื่อ Hover ต้องมีความรู้สึกเหมือนระบบกำลัง "Scan" หรือ "Audit" ข้อมูล
- **Custom Ease Functions**: ห้ามใช้ `ease-in-out` มาตรฐาน ให้ใช้ `cubic-bezier(0.19, 1, 0.22, 1)` เพื่อความพรีเมียม
- **Haptic Visuals**: การตอบสนองต้องดูมี "น้ำหนัก" (Weight) และ "ความหนืด" (Friction) ที่พอเหมาะ

## 🌐 4. Structural Data Environments

- **Audit-Grid System**: พื้นหลังต้องสื่อถึงโครงสร้าง (Grids, Crosshairs, Status Indicators)
- **High-Contrast Markers**: ใช้สี Accent (Gold/Emerald) เฉพาะจุดที่เป็นผลลัพธ์หรือความสำเร็จ เพื่อดึงสายตามนุษย์
