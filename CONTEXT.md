# 🧠 Business Context: JP-Visual&Docs (Strategic Service Provider)

## 1. Vision & Identity

เราไม่ใช่แค่เอเจนซี่ทำเอกสารทั่วไป แต่เราคือ "Strategic Solutions Provider" ที่ใช้ Logic และ System Design มาแก้ปัญหาที่ซับซ้อนด้านการเงิน (Financial), การย้ายถิ่นฐาน (Immigration) และโครงสร้างระบบ (Systems)

## 2. Target Audience (High-End & Hard Cases)

- **กลุ่มเป้าหมาย:** นักธุรกิจ, Freelance รายได้สูง, หรือผู้ที่ต้องการผลลัพธ์ระดับ 100% แต่มีข้อจำกัดด้านโปรไฟล์ (เช่น ที่มาเงินไม่ชัดเจน, เคยโดนปฏิเสธวีซ่า)
- **Pain Point:** ความไม่ชัดเจนของขั้นตอน, กลัวการถูกปฏิเสธ, เบื่อการบริการแบบบ้านๆ ที่ไม่มีมาตรฐาน

## 3. Service Philosophy (The "Unfair Advantage")

ข้อมูลที่เรามอบให้ต้องเป็น **"Deep Insights"** ที่หาไม่ได้ทั่วไป:

- **Financial:** ไม่ใช่แค่เดินบัญชี แต่คือ "Cashflow Engineering" และ "Internal Audit Simulation"
- **Immigration:** ไม่ใช่แค่จองตั๋ว แต่คือ "GDS System Injection" ที่ตรวจสอบสถานะได้จริงในระบบสากล
- **Documentation:** ไม่ใช่แค่พิมพ์งาน แต่คือ "Logic-based Profiling" ที่อุดทุกรอยโหว่ของการตรวจสอบ

## 4. Content Standard (The No-Fluff Rule)

- **No Basic Info:** ห้ามอธิบายสิ่งที่คนรู้อยู่แล้ว เช่น "วีซ่าคืออะไร" หรือ "ทำไมต้องมีสเตทเม้น"
- **High-Impact Insights:** ให้เจาะจงไปที่ "Algorithm การตรวจของเจ้าหน้าที่" หรือ "Logic หลังบ้านของระบบธนาคาร"
- **Language:** ภาษาไทยระดับ Professional Peer (คม, กระชับ, น่าเชื่อถือ เหมือนผู้เชี่ยวชาญคุยกับเจ้าของธุรกิจ)

## 5. Technical Stack

- Next.js 15 (App Router), TypeScript, Tailwind CSS, MDX
- เน้น Performance (SEO Score 100) และ Clean Code (SOLID Principles)
